# 0x08 - Recursion

This project covers recursive functions in C.

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | 0-puts_recursion.c | Prints a string using recursion |
| 1 | 1-print_rev_recursion.c | Prints a string in reverse using recursion |
| 2 | 2-strlen_recursion.c | Returns length of string using recursion |
| 3 | 3-factorial.c | Returns factorial of a number |
| 4 | 4-pow_recursion.c | Returns x raised to power y |
| 5 | 5-sqrt_recursion.c | Returns natural square root |
| 6 | 6-is_prime_number.c | Checks if number is prime |

## Compilation

gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c main.c function.c -o program
