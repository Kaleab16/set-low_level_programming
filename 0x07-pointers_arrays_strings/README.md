# 0x07 - Pointers, Arrays, and Strings (Advanced)

This project covers advanced pointer manipulation, memory functions, and string operations.

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | 0-memset.c | Fills memory with constant byte |
| 1 | 1-memcpy.c | Copies memory area |
| 2 | 2-strchr.c | Locates character in string |
| 3 | 3-strspn.c | Gets length of prefix substring |
| 4 | 4-strpbrk.c | Searches string for set of bytes |
| 5 | 5-strstr.c | Locates substring |
| 6 | 6-print_chessboard.c | Prints chessboard |
| 7 | 7-print_diagsums.c | Prints sum of diagonals |

## Compilation

gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c main.c function.c -o program
