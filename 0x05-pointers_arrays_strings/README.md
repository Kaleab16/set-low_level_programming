# 0x05 - Pointers, Arrays, and Strings

This project covers pointer manipulation, arrays, and string operations in C.

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | 0-reset_to_98.c | Updates value to 98 using pointer |
| 1 | 1-swap.c | Swaps two integers |
| 2 | 2-strlen.c | Returns length of string |
| 3 | 3-puts.c | Prints string followed by newline |
| 4 | 4-print_rev.c | Prints string in reverse |
| 5 | 5-rev_string.c | Reverses a string |
| 6 | 6-puts2.c | Prints every other character |
| 7 | 7-puts_half.c | Prints half of a string |
| 8 | 8-print_array.c | Prints n elements of array |
| 9 | 9-strcpy.c | Copies string |

## Compilation

gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c main.c function.c -o program
