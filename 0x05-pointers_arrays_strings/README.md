# 0x05-pointers_arrays_strings

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | `0-reset_to_98.c` | Updates value to 98 using pointer |
| 1 | `1-swap.c` | Swaps values of two integers |
| 2 | `2-strlen.c` | Returns length of a string |
| 3 | `3-puts.c` | Prints a string |
| 4 | `4-print_rev.c` | Prints string in reverse |
| 5 | `5-rev_string.c` | Reverses a string |
| 6 | `6-puts2.c` | Prints every other character |
| 7 | `7-puts_half.c` | Prints half of a string |
| 8 | `8-print_array.c` | Prints array elements |
| 9 | `9-strcpy.c` | Copies a string |

## Requirements
- All files compiled with: `gcc -Wall -pedantic -Werror -Wextra -std=gnu89`
- Betty style compliant
- No more than 5 functions per file
- No global variables
