# 0x04-more_functions_nested_loops

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | `0-isupper.c` | Checks for uppercase character |
| 1 | `1-isdigit.c` | Checks for a digit (0-9) |
| 2 | `2-mul.c` | Multiplies two integers |
| 3 | `3-print_numbers.c` | Prints numbers 0-9 |
| 4 | `4-print_most_numbers.c` | Prints numbers 0-9 except 2 and 4 |
| 5 | `5-more_numbers.c` | Prints numbers 0-14 ten times |
| 6 | `6-print_line.c` | Draws a straight line |
| 7 | `7-print_diagonal.c` | Draws a diagonal line |
| 8 | `8-print_square.c` | Prints a square |
| 9 | `9-fizz_buzz.c` | Fizz-Buzz test program |
| 10 | `10-print_triangle.c` | Prints a triangle |

## Requirements
- All files compiled with: `gcc -Wall -pedantic -Werror -Wextra -std=gnu89`
- Betty style compliant
- No more than 5 functions per file
- No global variables
- Only _putchar for output (except Task 9)
