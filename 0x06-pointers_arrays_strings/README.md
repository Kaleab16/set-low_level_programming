# 0x06 - Pointers, Arrays, and Strings

This project covers string manipulation functions similar to the C standard library.

## Tasks

| Task | File | Description |
|------|------|-------------|
| 0 | 0-strcat.c | Concatenates two strings |
| 1 | 1-strncat.c | Concatenates two strings using at most n bytes |
| 2 | 2-strncpy.c | Copies a string |
| 3 | 3-strcmp.c | Compares two strings |
| 4 | 4-rev_array.c | Reverses an array of integers |
| 5 | 5-string_toupper.c | Converts lowercase to uppercase |
| 6 | 6-cap_string.c | Capitalizes all words |
| 7 | 7-leet.c | Encodes string into 1337 (leet speak) |

## Compilation

gcc -Wall -pedantic -Werror -Wextra -std=gnu89 main.c function.c -o program
